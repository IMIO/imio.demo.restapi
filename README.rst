Installation & Test
-------------------

Run demo instances::

    make iadelib-up
    make urban-up
    make middleware-up

The fake IA Delib instance will run on: http://localhost:8080
The fake URBAN instance will run on: http://localhost:8081
The middleware instance will run on: http://localhost:6543
